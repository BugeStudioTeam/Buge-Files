## ✨ Features

- 📁 **Full File Browsing** — Navigate your entire internal storage freely
- 🔍 **Real-time Search** — Recursively search files and folders from any directory
- ⭐ **Favorites** — Bookmark frequently accessed files and folders
- 🕐 **Recent Files** — Quickly access your most recently modified files
- 🌗 **Light / Dark / System Theme** — Full Material You theming support
- 🗂️ **List & Grid View** — Switch between views with one tap
- 📝 **Create Files & Folders** — Create new files or folders directly in the app
- ✏️ **Rename & Delete** — Manage your files with ease
- 📤 **Share Files** — Share any file to other apps instantly
- 🔒 **Hidden Files Toggle** — Show or hide dotfiles from settings
- 🌍 **Multi-language Support** — English and Chinese (Simplified) included
- 📦 **File Type Icons** — Distinct icons for images, video, audio, PDF, APK, archives and more
- 🔃 **Sort Options** — Sort by name, date, or size in ascending or descending order
- 📂 **Set as Default File Manager** — Register as the system file manager app

## 🔐 Permissions

| Permission | Purpose |
|---|---|
| `READ_EXTERNAL_STORAGE` | Read files on Android 6–12 |
| `WRITE_EXTERNAL_STORAGE` | Write files on Android 6–9 |
| `READ_MEDIA_IMAGES / VIDEO / AUDIO` | Read media files on Android 13+ |
| `MANAGE_EXTERNAL_STORAGE` | Full storage access on Android 11+ |

> On Android 11 and above, the app will redirect you to the system settings page to grant **All Files Access** permission.
